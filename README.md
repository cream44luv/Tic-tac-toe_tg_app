Мультиплеерная игра в крестики-нолики, интегрированная с Telegram в формате Mini App. 
Приложение позволяет играть с друзьями, отслеживать их онлайн-статус и отправлять приглашения прямо в чат.

## ✨ Особенности

- 🎯 **Чистый интерфейс** — адаптирован под темы Telegram (светлая/темная)
- 🔵 **Синие крестики** и 🔴 **красные нолики** — сочная цветовая схема
- 👥 **Социальные функции**:
  - Список всех игроков с поиском по нику
  - Отдельная вкладка "Друзья" с онлайн-статусом
  - Приглашения через бота
- 📱 **Адаптивный дизайн** — отлично выглядит на всех устройствах
- ⚡ **Быстрый старт** — никакой регистрации, вход через Telegram

## 🛠️ Технологии

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Telegram Integration**: Telegram Web App SDK
- **Backend**: Node.js, Express
- **Bot**: node-telegram-bot-api

## 📋 Требования

- Node.js 14+
- Telegram бот (создается через @BotFather)
- Аккаунт на GitHub (для хостинга статики)
